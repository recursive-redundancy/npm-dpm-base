(inner) stripValue(value, validator) → {string|null}

Helper function for start of all module conversion functions. Takes input value and strips it down. First, Checks supplied module's isValid function to test value is valid. Then, converts value to string and validates the value according to base format. Finally, trims value for leading zeroes, leaving a properly stripped value to convert.
